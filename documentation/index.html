Description

Api para gerenciamento de jogos, onde é possível se cadastrar, obter informações sobre jogos, e adicionar os favoritos em um perfil pessoal.

Installation

$npm install

##Security

$npm i bcrypt

$npm i -D @types/bcrypt

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

Test

# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

Swagger (Test)

http://localhost:3333/api/
